About this Application

This is an application submission for Riot's First API Challenge. It is a two week challenge that involves using game data from U.R.F matches and using various endpoints provided by Riot to create something (anything!).

This web application uses a MEAN stack and comprises of both a REST back end application and an Angular front end application using Bootstrap. I followed the MEANJS example and highly encourage those interested in full webstack development in JS to "check it out" ;) . The application's database is hosted externally via MongoLab and I highly recommend them as well.

Technologies Used:

  • MongoDB
  • ExpressJS
  • AngularJS
  • NodeJS
  • Bootstrap
  • MongooseJS
